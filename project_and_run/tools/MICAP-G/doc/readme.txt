Ver.2.0からの変更点について
------------------------------

　Ver.2.0からVer.2.1への変更点は以下の通りです。

- メインウインドウ
　- Mapパラメータを設定したとき、自動的にFieldパラメータを計算し、
　　図示する機能を追加しました。
　- ウインドウ内の地図描画位置をSEIS95・PAT-ME互換に変更しました。
　- SEIS95・PAT-ME互換の地形描画（等高線・色分け）機能を追加しま
　　した。

- Mapパラメータ設定ダイアログの変更
　- プレビュー画面を追加しました。
　- 地図エリアの追加・削除機能を追加しました。
　- 地形描画設定チェックボックスを追加しました。

- 計算結果出力ファイル種類の追加
　- 格子点・観測点の両方について、面積ひずみ、体積ひずみ、最大せ
　　ん断ひずみ、△CFFの各値を出力できるようにしました。

- バグの修正
　- micapg.iniにassist3formの情報が保存されないバグを修正しまし
　　た。
　- カラープリンタで印刷したとき、火山位置マークが黒になるバグを
　　修正しました。
　- カラープリンタで印刷したとき、コンターの凡例が黒になるバグを
　　修正しました。
　- Visual設定ダイアログをCancelボタンで閉じたときに、メインウイ
　　ンドウにスケール軸が表示されたままになるバグを修正しました。 
　- Fieldパラメータ設定ダイアログで、Inclinedチェックボックスに
　　ついてCancelが効かないバグを修正しました。
